# Image-stegnography-lsb

This Project implements image stegnography using Least Significant Bit (LSB) technique in C.
It allows hiding a secret message inside an image and retriving it back without noticeable changes to the image.

## Features:-
- Encode secret message into an image.
- Decode hidden message from image
- uses LSB technique for data hiding

  ## Concepts Used:-
  - File Handling
  - Bitwise Operations
  - Structures
  - Pointers
 
    ## How it works:-
    The least significant bits of image pixels are modified to store the secret message data.
